# Changelog

## 0.2.0 — 2026-07-13

- Added compatible unique-index selection, including compound and partial unique indexes.
- Added automatic partial-filter handling for soft-deleted records.
- Added a searchable collection picker and migration-direction switching.
- Added saved comparison configuration and an activity log for progress and errors.
- Added migration-wide field and subdocument exclusions.
- Added full side-by-side, code-review-style document diffs with insert, modify, and delete highlighting.
- Added detailed migration history with document actions, filters, selected fields, and schema-operation review.
- Added confirmation-gated document rollback for completed migrations.
- Fixed repeat-comparison crashes and improved full-window layout behavior.
- Fixed app icon metadata for the built macOS bundle.

## 0.1.0

- Initial native macOS application and local MCP server.
